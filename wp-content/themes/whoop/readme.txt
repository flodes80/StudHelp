== Description ==
A Child Theme for Directory Starter, built to be used with the GeoDirectory plugin to create a Yelp like directory website.
Create your very own yelp like directory site for your city or global directory of your niche category, be it restaurants, bar or museums.

== Frequently Asked Questions ==

= Is the GeoDirectory plugin required? =

No, this is optional but recommended.
